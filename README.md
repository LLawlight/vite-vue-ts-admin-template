# vue-typescript-admin-template

## 总览

这是一个极简的 vue typescript vite admin 管理后台。它只包含了 Element3 & axios & svgicon & permission control & lint，这些搭建后台必要的东西。

[electron-vue-admin](https://github.com/PanJiaChen/electron-vue-admin)

## 如何设置以及启动项目

### 安装依赖

```bash
npm install
```

### 启动本地开发环境

```bash
npm run dev
```

### 构建生产环境

```bash
npm run build
```

### 预览发布环境效果

```bash
npm run serve
```

### 代码格式检查以及自动修复

```bash
npm run lint
```
